The first agent testing framework built around deterministic contracts.

Stop scoring your agent's output quality using probabilistic vibes. Start enforcing precisely what your agent MUST do, MUST NOT do, and in what order, using 100% deterministic code.

test_booking.py 
from agentcontract import contract, ContractRunner

@contract("flight_booking")
async def test_books_correct_flight():
    runner = ContractRunner(adapter="openai")
    result = await runner.arun(agent=my_agent, input="Book flight")

    # The Deterministic Contract
    result.must_call("search_flights")
    result.must_call("book_flight").after("search_flights")
    result.must_call("book_flight").with_args(destination="NYC")
    result.must_not_call("cancel_booking")
    
    # Advanced Profiling
    result.assert_total_duration_under(ms=5000)
    
    # Record Golden Trajectory
    result.snapshot()

Why AgentSpec?

🛡️

100% Deterministic

No LLMs-as-judges giving you a random 0.8 passing score. AgentSpec assertions either pass or logically fail.

🕵️

Multi-Agent Tracking

Ensure isolated agents execute precise tools strictly independently by asserting behaviors using trace tracking IDs.

⏱️

Performance Boundaries

Strictly bounded SLAs. Restrict full execution loops or specify exact timeout bounds for singular external tool traces.

🌪️

Chaos Engine Testing

Simulate production errors. Force tools to hit RateLimits, drop databases, or inject random stochastic latencies to verify application retry patterns.
